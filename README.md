# 🛡️ Enterprise SOC Home Lab

## About This Project

I'm building this Enterprise SOC Home Lab to gain practical experience in Windows Server administration, Active Directory, Blue Team operations, and enterprise security.

Instead of using a pre-built environment, I'm creating the lab from scratch to understand how enterprise infrastructures are designed, configured, and secured. Every stage of the project is documented to track my learning and create a reference for future cybersecurity projects.

This lab will continue to evolve as I learn new technologies and implement more advanced security solutions.
